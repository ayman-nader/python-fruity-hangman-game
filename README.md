# 🎮 Hangman Game (Python)

This project is a command-line Hangman game built using Python as part of my Data Science training course at Epsilon AI.

## 🚀 Features
- Interactive gameplay
- Dynamic feedback messages
- Word selection logic
- Error handling and input validation
- Lighthearted UI
- Random interactive messages
- Different difficulties
- Dynamic attempt counter based on difficulty choice
- Replay option (within loop and respectful of DRY principal)
- Other non-implemented bonus ideas: (Hint system, Visual progression system)

## 🧠 Skills Demonstrated
- Python fundamentals
- Control flow (loops, conditions)
- Functions and modular code
- Error handling
- Problem-solving & logic building
